# Zamblak UI RTL & Senior UX Guard

**When to use:** During front-end implementation and styling.
**What it guards:** Arabic RTL layout, Tailwind best practices, Next.js Server Actions integration, and Stitch workflow alignment.
**Required checks:** Ensure logical properties are used for RTL (`ms`, `me`, `ps`, `pe`).
**Forbidden actions:** Using Google Stitch as a source of truth for business logic.
**PASS/HOLD expectations:** UI components are accessible, responsive, and RTL-ready.
